User IP - Simply display visitor IP Address on post or page with a shortcode.

== Description ==

This plugin simply display IP Address on post or page with a shortcode.

Very simple to install, simple to use.

== Installation ==

1. Upload `user-ip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode [user_ip] in your page or post

== Frequently Asked Questions ==

= Is User IP free of charge ? =

Yes, of course.

= How to use this plugin properly ? =

Intall it. Activate it.
Then use this shortcode in your page or post :

[user_ip]

= Can I show User IP Address in a sidebar widget ? =

Yes, it works in a "text widget" with this same shortcode [user_ip].

= Where is the plugin settings ? =

There is no settings.

= Can I show Server IP Address or Admin IP Address with this plugin ? =

No, you can't. This plugin only show user IP Address.
To display Server IP Address, use this WordPress plugin :
http://wordpress.org/plugins/server-ip/
